## 프로젝트 스토리

### 이 앱은 어디서 왔나요?

제 아내는 코바늘 작가입니다.

번아웃을 이겨내고 다시 뜨개바늘을 잡았을 때, 도안을 그릴 적당한 도구가 없었습니다.
엑셀은 너무 딱딱하고, Procreate는 뜨개 기호를 모르고, 전문 소프트웨어는 수십만 원입니다.

그래서 남편인 제가 직접 만들기 시작했습니다.
"멈춰도 돼, 포기만 안 하면"이라는 아내의 슬로건처럼, 완벽하지 않아도 계속 나아가는 도구를.

### Crochet Flow가 다른 이유

**뜨개를 아는 앱입니다.**
코바늘 18종·대바늘 20종 표준 기호와 주석 기호 13종이 기본 탑재되어 있습니다. 사슬코, 짧은뜨기, 한길긴뜨기, 겉뜨기, 안뜨기—산업 표준 기호를 그대로 스탬프처럼 찍을 수 있습니다.

**Procreate처럼 씁니다.**
레이어, 대칭 그리기, 컬러휠, 핀치·줌·회전 제스처, QuickMenu, 레이어 마스크, 선택 영역 변형—뜨개 작가가 앱 사용법을 새로 배울 필요가 없습니다.

**도안 작업 전용 기능이 있습니다.**
격자 스냅으로 기호가 자동 정렬되고, 치수선 도구로 칸수를 표시하고, 단 계산기 HUD가 화면 한켠에 항상 떠 있습니다. 서술도안 자동 생성 후 PDF로 바로 인쇄할 수 있습니다.

**웹·앱·오프라인 모두 됩니다.**
브라우저에서 바로 열리고, 홈화면에 추가하면 앱처럼 동작하며, 와이파이 없어도 작동합니다. Android·iOS 네이티브 앱은 이번 펀딩으로 개발합니다.

## 주요 기능 스크린샷 설명 (5장 기준)

1. **메인 캔버스** — 격자 위에 기호 스탬프 + 레이어 패널
2. **팔레트 탭** — 코바늘·대바늘·기타·자작기호·주석 5탭
3. **QuickMenu + 대칭** — 방사·수직·수평 대칭, 6슬롯 커스텀 메뉴
4. **서술도안 생성** — 레이어별 기호 카운트 → 한국어 서술 → PDF
5. **단 계산기 + 치수선** — 우하단 HUD 단 카운터, 2점 클릭 치수 레이블

## FAQ

**Q. iOS·Android 앱은 언제 나오나요?**
A. 펀딩 성공 후 3개월 내 베타 출시를 목표로 합니다. 현재 웹앱(PWA)으로 홈화면 추가 후 앱처럼 사용 가능합니다.

**Q. 기존 도안 파일을 가져올 수 있나요?**
A. .cony 프로젝트 파일 형식을 지원합니다. PNG/JPG 이미지도 참조 이미지로 불러올 수 있습니다.

**Q. 오프라인에서 사용 가능한가요?**
A. 네. 한 번 열면 Service Worker가 앱을 캐시해 와이파이 없이도 작동합니다.

**Q. 뜨개 기호를 직접 만들 수 있나요?**
A. Pro 플랜에서 자작 기호 에디터를 지원합니다. SVG 경로를 그려 팔레트에 추가할 수 있습니다.

**Q. 결제 보안은 안전한가요?**
A. 결제는 토스페이먼츠(금감원 등록 PG사)를 통해 처리됩니다. 카드 정보는 저희 서버를 거치지 않습니다.

**Q. 환불 정책은?**
A. 미사용 기간에 대한 일할 환불을 지원합니다. 문의: cjh951013@gmail.com
